 Project Overview
This project is a full-stack Flower Shop Management System developed using React, Node.js, Express, and MongoDB.

 Features
- Add Flower
- View Flowers
- Update Flower
- Delete Flower
- User Login and Register
- Profile Update

 Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Version Control: GitHub

Project Structure
- backend/
- frontend/

 How to Run the Project

 Backend
cd backend
npm install
npm run dev

 Frontend
cd frontend
npm install
npm start
